# leetcode_sql
# 📊 LeetCode SQL 练习记录

## 🧭 项目简介

本项目用于记录我在 LeetCode 上的 SQL 练习过程，作为转行成为数据分析师学习路径的一部分。通过系统化的刷题与总结，提升数据提取、处理与分析能力，建立扎实的 SQL 基础。

---

## 📅 学习计划

* **第1-2周**：每日刷题 20 道，覆盖基础语法、JOIN、GROUP BY、子查询等核心内容。
* **第3-4周起**：每天保持 4 道题巩固，重点提升窗口函数、CTE、性能优化等能力。
* 每周进行一次总结与回顾。
